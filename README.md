# The Gearsh App — MVP Demo Submission

## Overview
The Gearsh App is an innovative platform connecting local artists with bookers for seamless event talent management. This MVP demo showcases core features including artist discovery, profiles, and mock booking flows, all running offline with static data. (50 words)

## Features in this build
- Splash screen with MVP Demo badge
- Home dashboard (Discover page) with artist listings
- Artist profile pages with details and booking options
- Mock booking flow: profile → book artist → confirmation (snackbar notification)
- Navigation bar with branding colors and logo
- Offline functionality with static demo data

## Installation steps
1. Download the APK file: `gearsh_mvp_demo.apk`
2. Enable "Install from unknown sources" in Android settings
3. Install the APK on an Android device (API 29+ recommended)
4. Launch the app and tap "Enter Demo" on the splash screen
